# 在线视频课程系统

### 技术栈

#### 前端：Vue,Bootstrap,FreeMarker

#### 后端：Spring Cloud,MySQL,MyBatis Generator,Git,Maven



### 技术亮点

- #### 使用代码生成器mybatis-generator,freemarker快速开发业务模块

- #### 视频文件实现分片上传，断点续传

- #### 使用aop实现日志打印
